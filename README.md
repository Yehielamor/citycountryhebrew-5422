# citycountryhebrew-5422

CityCountryHebrew is a real-time multiplayer word game where players compete to name cities, countries, and other categories starting with a given Hebrew letter within a time limit. The MVP focuses on a simple, engaging interface for Hebrew speakers, featuring live scoring, category selection, and a leaderboard to foster competition and fun.

## Tech Stack
Next.js, Node.js with Socket.io, PostgreSQL, Tailwind CSS

## Setup
Run `docker-compose up` to start the infrastructure.
